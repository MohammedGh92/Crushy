Product: Match3 candy set
Version: 1.0

This Unity asset package was created to assist developers to create Match3
games. It contains .png textures for 64 different sweets in 6 colors in 64x64,
128x128 and 256x256 resolutions, including original .ai file to save in any other resolution.
Also, it contains ready animated prefabs, including animation of landing, shake, glow,
hint and destroy.
